# Student CRUD Application

This is a comprehensive Student CRUD (Create, Read, Update, Delete) application.

## Overview

This repository contains the source code for managing student records. It demonstrates fundamental operations typically found in database-backed applications.

## Features

- **Create:** Add new student records.
- **Read:** View existing students.
- **Update:** Modify student details.
- **Delete:** Remove student records.

## Setup

1. Clone the repository
2. Install dependencies
3. Run the application
